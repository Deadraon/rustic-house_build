# Rustic House - Cafe | Lounge | Bar

A modern, premium, fully responsive restaurant website for Rustic House in Roorkee.

## Structure

- `index.html` - Complete landing page markup
- `styles.css` - Custom rustic-luxury visual system and responsive styling
- `script.js` - Navbar behavior, mobile menu, scroll reveals and menu filtering
- `public/images/` - Cropped local image assets used across hero, gallery and backgrounds

## Run Locally

Open `index.html` directly in a browser, or run:

```bash
npm run dev
```

## Deploy

This is a static site and can be deployed directly to Vercel from the project root.
